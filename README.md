# Codes for the Cool Brown Dwarf JWST Survey

My goal is to not create a billion repos this time and maintain this one properly for the entire planning and analysis...
Let's see how it goes 🙃.

## Structure

- The notebooks are in `notebooks/`.
- Static resource files are in `resources/`
- Importable source code is in `src/coldest`

## Installing

First, clone the repo and `cd` into it

```bash
git clone https://github.com/vandalt/coldest.git
cd coldest
```

To be able to import code from `src/coldest`, create a virtual environment and run.

```bash
python -m pip install -U -e .
```
